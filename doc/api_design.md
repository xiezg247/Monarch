#### 1. 数据分类模板

##### 创建分类模板

```
POST /api/v1/template
```

###### 请求参数

| 字段名 | 类型   | 必填 | 长度 | 备注                       |
| ------ | ------ | ---- | ---- | -------------------------- |
| name   | string | 是   |      | 情感分析/图像识别/文本分类 |
|        |        |      |      |                            |

###### 请求参数格式

```
{
    "name": "情感分析",
}
```

###### 返回成功

```
{
    "code": 200,
    "msg": 提示,
    "data": {},
    "success": True
}
```

###### 返回失败

```
{
    "code": 400,
    "msg": 提示,
    "data": {},
    "success": False
}
```
